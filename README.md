***

### [Simpson's 1/3 Rule](github.com/katie-plese/MECH-105)
<dl>

</dl>

### [LU Factorization](github.com/katie-plese/MECH-105)
<dl>

</dl>

</dl>

### [Incremental Search](github.com/katie-plese/MECH-105)
